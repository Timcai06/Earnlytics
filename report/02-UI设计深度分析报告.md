### 4. 阴影和发光效果过度使用

**问题描述**: 过度使用阴影和发光效果，造成视觉噪音，降低内容可读性。

**问题代码**:
```typescript
// ❌ 过度装饰
<h1 className="drop-shadow-[0_0_30px_rgba(99,102,241,0.25)]">
<button className="shadow-[0_4px_20px_rgba(99,102,241,0.6)]">
<div className="shadow-[0_0_30px_rgba(99,102,241,0.25)]">
<span className="drop-shadow-[0_0_20px_rgba(99,102,241,0.38)]">
```

**设计原则**:
- 阴影应用于可交互元素（按钮、卡片悬停）
- 文字不应有阴影，会降低可读性
- 发光效果应谨慎使用，仅用于强调

**修复方案**:
```typescript
// ✅ 使用设计令牌中的标准阴影
<button className="shadow-glow-sm hover:shadow-glow-md transition-shadow">
<h1 className="text-white"> // 移除文字阴影
```

**修复优先级**: P1  
**预估工作量**: 2-3小时
