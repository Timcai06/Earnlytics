## 🛠 技术栈

- **前端**: Next.js 14 + Tailwind CSS + shadcn/ui
- **后端**: Vercel Serverless
- **数据库**: Supabase PostgreSQL
- **AI**: DeepSeek API
- **数据源**: FMP API + SEC EDGAR
- **自动化**: GitHub Actions
