# 数据脚本清单（你会常用哪些）

## 在哪里

`earnlytics-web/scripts/ops`

## 最常用命令

1. `npm run fetch:earnings`：抓财报
2. `npm run analyze:batch`：批量 AI 分析
3. `npm run alerts:process`：处理预警规则
4. `npm run sync:valuation`：估值数据同步

## 两个最核心脚本

1. `fetch-earnings.ts`
   - 从 FMP 拉数据
   - upsert 到 `earnings`
2. `analyze-batch.ts`
   - 找未分析财报
   - 调 AI
   - 写入 `ai_analyses`

## 新手注意

1. 脚本使用 `.env.local`，没有 key 会失败
2. 部分脚本会写数据库，执行前要确认环境
3. 先在测试数据或开发环境跑
