# 你将学到什么

## 项目一句话

Earnlytics 是一个 AI 财报分析网站：

1. 抓取美股公司财报
2. 用 AI 生成中文分析
3. 在网页展示给用户

## 技术栈（新手版）

1. Next.js：网站框架（页面 + API）
2. React：前端组件
3. TypeScript：带类型的 JavaScript
4. Supabase：数据库服务
5. Vercel：部署平台

## 你现在不需要担心的事

1. 不需要一次性看完所有代码
2. 不需要先学完全部前端理论
3. 不需要先学复杂算法

## 你现在必须做的事

1. 先跑起来项目
2. 能看懂请求从页面到 API 的路径
3. 能读懂最基础的 TS 语法
