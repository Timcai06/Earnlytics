# 本地启动步骤（一步一步做）

## 第 1 步：进入项目

```bash
cd /Users/justin/Desktop/earnlytics/earnlytics-web
```

## 第 2 步：安装依赖

```bash
npm install
```

## 第 3 步：配置环境变量

```bash
cp .env.example .env.local
```

然后编辑 `.env.local`，填上真实 key。

## 第 4 步：启动开发环境

```bash
npm run dev
```

## 第 5 步：浏览器验证

打开：

1. `http://localhost:3000/`
2. `http://localhost:3000/home`
3. `http://localhost:3000/companies`
4. `http://localhost:3000/api/health`

## 常见报错速查

1. `Cannot find module`：通常是依赖没装好，重新 `npm install`
2. `Supabase not configured`：环境变量未配置
3. 端口占用：换端口 `npm run dev -- --port 3001`
