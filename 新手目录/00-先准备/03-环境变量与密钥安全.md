# 环境变量与密钥安全

## 文件位置

1. 示例文件：`earnlytics-web/.env.example`
2. 本地文件：`earnlytics-web/.env.local`

## 核心变量说明

1. `NEXT_PUBLIC_SUPABASE_URL`：Supabase 地址
2. `NEXT_PUBLIC_SUPABASE_ANON_KEY`：前端可见 key
3. `SUPABASE_SERVICE_ROLE_KEY`：服务端高权限 key
4. `FMP_API_KEY`：财报数据源
5. `DEEPSEEK_API_KEY`：AI 分析

## 必须记住的安全规则

1. `SUPABASE_SERVICE_ROLE_KEY` 只能在服务端/API/脚本使用
2. 不要把 `.env.local` 提交到 Git
3. `NEXT_PUBLIC_` 开头变量默认可被前端访问

## 如何快速检查配置是否生效

1. 启动项目 `npm run dev`
2. 打开 `http://localhost:3000/api/health`
3. 看响应里的 `env` 和 `tests` 字段
