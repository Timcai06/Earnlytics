# 一次请求的完整旅程

## 场景 A：访问页面 `/home`

```text
浏览器请求 /home
-> src/app/home/page.tsx (服务端页面)
-> src/app/home/home-data.ts (查询数据库和行情)
-> 把数据传给 src/app/home/HomePageClient.tsx
-> 浏览器渲染页面
```

关键点：

1. `page.tsx` 负责组织数据
2. `HomePageClient.tsx` 负责交互和展示
3. 页面不是直接查数据库，而是通过服务逻辑汇总数据

## 场景 B：访问接口 `/api/companies`

```text
浏览器或前端 fetch('/api/companies')
-> src/app/api/companies/route.ts
-> src/lib/supabase.ts
-> 查询 companies 表
-> NextResponse.json 返回
```

关键点：

1. API 是后端入口
2. `route.ts` 里可以连数据库
3. 返回的是 JSON，不是 HTML
