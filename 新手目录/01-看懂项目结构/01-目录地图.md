# 目录地图（先知道每层做什么）

## 根目录

`/Users/justin/Desktop/earnlytics`

这里有多个内容，但你开发时主要看 `earnlytics-web`。

## 网站主目录

`earnlytics-web` 里的关键结构：

1. `src/app`：页面与 API
2. `src/components`：UI 组件
3. `src/lib`：数据库、AI、工具函数
4. `src/types`：类型定义
5. `scripts/ops`：数据更新脚本
6. `supabase/migrations`：数据库 SQL

## 记忆口诀

1. `app` 决定“入口在哪里”
2. `components` 决定“页面怎么拼”
3. `lib` 决定“功能怎么实现”
4. `scripts` 决定“数据怎么进来”
