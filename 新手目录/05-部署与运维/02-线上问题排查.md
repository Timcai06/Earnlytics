# 线上问题排查清单

## 问题 1：页面 500

1. 看 Vercel 日志
2. 看对应 API route 的报错
3. 检查环境变量是否丢失

## 问题 2：接口返回空数据

1. 检查数据库是否有数据
2. 检查脚本任务是否成功执行
3. 检查查询条件是否过严

## 问题 3：AI 功能失败

1. 检查 `DEEPSEEK_API_KEY`
2. 查看 `src/lib/ai.ts` 报错信息
3. 检查外部 API 配额与网络

## 问题 4：股价接口不稳定

1. 看 `/api/stock-price/[symbol]` 返回 `source`
2. 区分 cache/live/unavailable
3. 排查外部行情接口是否波动
