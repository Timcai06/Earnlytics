# 动手练习与验收标准

## 练习 1：改首页文案

操作：修改 `src/components/home/LandingPageUI.tsx` 的标题。

验收标准：

1. 页面显示新文案
2. 无 TypeScript 报错
3. `npm run build` 通过

## 练习 2：改 API 查询字段

操作：把 `src/app/api/companies/route.ts` 的 `select('*')` 改为更精简字段。

验收标准：

1. `/api/companies` 返回结构变化正确
2. 前端若依赖该字段，能同步修复

## 练习 3：新增一个最小 API

操作：新增 `src/app/api/hello/route.ts`，返回 `{ ok: true }`。

验收标准：

1. 浏览器访问 `/api/hello` 可返回 JSON
2. 代码风格与现有 route.ts 一致

## 练习 4：写一个类型安全格式化函数

目标：输入 `number | null`，输出可展示货币字符串。

验收标准：

1. `null` 返回 `N/A`
2. number 正常格式化
3. 函数参数和返回值都有类型标注
