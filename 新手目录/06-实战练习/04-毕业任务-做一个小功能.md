# 毕业任务：做一个小功能（通过即“入门完成”）

## 任务目标

新增一个“股票代码查询小工具”页面：

1. 页面路径：`/tool/symbol-check`
2. 输入股票代码
3. 调用 `/api/stock-price/[symbol]`
4. 展示价格与涨跌

## 要求

1. 使用 TypeScript 定义返回类型
2. 处理加载态和错误态
3. 处理空输入
4. UI 保持简洁清晰

## 验收标准

1. `npm run lint` 通过
2. `npm run build` 通过
3. 手动测试 AAPL/MSFT 两个代码可用
4. 无 `any`、无 `ts-ignore`

## 你会因此掌握

1. 新建页面
2. 调用 API
3. 处理异步数据
4. 编写类型安全前端逻辑
