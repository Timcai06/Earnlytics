# 常见错误清单（新手高频）

## 错误 1：忘记 `use client`

现象：在页面里用了 `useState` 但报错。

修复：在文件第一行加 `"use client";`。

## 错误 2：类型写成 `any`

现象：代码暂时跑得过，但后续 bug 多。

修复：优先补 `type/interface`，至少先写联合类型。

## 错误 3：直接信任后端返回

现象：运行时报 `undefined` 或 `null`。

修复：使用 `?.` 和 `??`，对空值做兜底。

## 错误 4：把 service role key 用在前端

现象：安全风险极高。

修复：只在服务端 API 或脚本中使用 `SUPABASE_SERVICE_ROLE_KEY`。

## 错误 5：看不懂路由映射

修复：记住规则：

1. 页面是 `page.tsx`
2. API 是 `route.ts`
3. 动态段是 `[param]`
