# 公司列表链路（/companies）

## 入口文件

1. 服务端页面：`src/app/companies/page.tsx`
2. 客户端组件：`src/app/companies/CompaniesList.tsx`

## 数据流程

```text
page.tsx
-> 查询 companies
-> 批量查 earnings 获取每家公司最新财报
-> 组装 CompanyWithEarnings[]
-> 传给 CompaniesList.tsx
-> 前端执行搜索、筛选、排序
```

## 你在这个页面能学到

1. 服务端数据预处理
2. 前端 useMemo 优化列表计算
3. 搜索和筛选如何协作

## 推荐练习

1. 给筛选增加一个行业
2. 增加一个排序项（例如按公司名）
3. 确认类型定义仍通过
