# 新手目录（Earnlytics 学习路径）

> 适合：只有一点计算机基础，希望系统看懂这个项目的人。

## 学习目标

学完本目录后，你应该能：

1. 独立把项目跑起来并定位常见错误
2. 看懂页面、API、数据库、脚本之间如何协作
3. 读懂项目里的 TypeScript/TSX 代码并做小改动
4. 理解项目如何部署到线上并持续更新数据

## 建议学习顺序

1. `00-先准备`
2. `01-看懂项目结构`
3. `02-语言基础`
4. `03-核心业务链路`
5. `04-数据与数据库`
6. `05-部署与运维`
7. `06-实战练习`

## 每天学习节奏（建议）

1. 每天 30-60 分钟
2. 每天只读 1-2 个文件
3. 每个文件读完后，至少做 1 个小练习

## 关键入口文件（先记住）

1. `earnlytics-web/package.json`
2. `earnlytics-web/src/app/layout.tsx`
3. `earnlytics-web/src/app/page.tsx`
4. `earnlytics-web/src/app/home/page.tsx`
5. `earnlytics-web/src/app/api/companies/route.ts`
6. `earnlytics-web/src/lib/supabase.ts`
7. `earnlytics-web/scripts/ops/fetch-earnings.ts`

## 学习方式说明

本目录的文档是“从外到内”的结构：

1. 先理解网站整体怎么跑
2. 再理解代码语法
3. 再理解业务链路
4. 最后做练习验证
