## Week 3：新功能开发

### 财报日历页面

**创建页面** (`app/calendar/page.tsx`)：

```typescript
export default async function CalendarPage() {
  // 获取未来30天的财报日历
  const { data: calendar } = await supabase
    .from('earnings_calendar')
    .select('*, companies(symbol, name)')
    .gte('date', new Date().toISOString())
    .order('date', { ascending: true })
    .limit(20)
  
  return (
    <div className="container mx-auto p-4">
      <h1 className="text-2xl font-bold mb-6">财报日历</h1>
      
      <div className="grid gap-4">
        {calendar?.map((item) => (
          <div key={item.id} className="border p-4 rounded">
            <div className="flex justify-between">
              <h3>{item.companies.name} ({item.companies.symbol})</h3>
              <span>{new Date(item.date).toLocaleDateString('zh-CN')}</span>
            </div>
            <p>预期EPS: ${item.eps_estimate}</p>
          </div>
        ))}
      </div>
    </div>
  )
}
```

### 邮件订阅系统

**1. 注册Resend** (resend.com)
- 免费额度：100封/天
- 获取API Key

**2. 添加订阅表**：

```sql
CREATE TABLE subscribers (
  id SERIAL PRIMARY KEY,
  email VARCHAR(255) UNIQUE NOT NULL,
  is_active BOOLEAN DEFAULT TRUE,
  subscribed_at TIMESTAMP DEFAULT NOW()
);
```

**3. 创建订阅组件**：

```typescript
// components/SubscribeForm.tsx
'use client'

import { useState } from 'react'

export function SubscribeForm() {
  const [email, setEmail] = useState('')
  
  async function handleSubmit(e: React.FormEvent) {
    e.preventDefault()
    
    await fetch('/api/subscribe', {
      method: 'POST',
      body: JSON.stringify({ email })
    })
    
    alert('订阅成功！')
    setEmail('')
  }
  
  return (
    <form onSubmit={handleSubmit} className="flex gap-2">
      <input
        type="email"
        value={email}
        onChange={(e) => setEmail(e.target.value)}
        placeholder="输入邮箱订阅财报提醒"
        className="border p-2 rounded flex-1"
        required
      />
      <button type="submit" className="bg-blue-500 text-white px-4 py-2 rounded">
        订阅
      </button>
    </form>
  )
}
```
