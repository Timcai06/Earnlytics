# 项目简介

> Earnlytics 项目完整介绍

---

## 是什么

Earnlytics是一个**自动化**的美国科技公司财报分析网站：

1. **自动获取**财报数据（FMP API + SEC EDGAR）
2. **AI自动分析**（DeepSeek生成中文摘要）
3. **自动发布**（GitHub Actions + Vercel ISR）
4. **被动收入**（Google AdSense展示广告）

## 为什么做

**市场痛点**：
- 财报数据太专业，普通投资者看不懂
- 中文财报分析资源少、更新慢
- 现有平台要么太慢，要么太复杂

**解决方案**：
- AI快速生成易读的中文摘要
- 专注科技巨头（最热门）
- 财报发布后1小时内上线

## 怎么赚钱

```
前端：Next.js 14 + Tailwind CSS
后端：Vercel Serverless（免费）
数据库：Supabase PostgreSQL（免费）
AI：DeepSeek API（¥0.002/千tokens）
自动化：GitHub Actions（免费）
```

## 成功关键因素

1. **速度** - 财报发布后越快上线，SEO排名越好
2. **质量** - AI分析准确、易读
3. **覆盖** - 足够多公司，满足多样化需求
4. **SEO** - 自然搜索是主要流量来源

## 风险提示

- **数据源风险**：API服务中断（有多数据源备份）
- **AI质量风险**：分析质量波动（持续优化Prompt）
- **SEO风险**：排名波动（专注长尾关键词）
- **合规风险**：金融信息需谨慎（明确免责声明）
