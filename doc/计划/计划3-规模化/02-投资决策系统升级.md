## 📊 投资决策核心维度

### 维度1: 基本面分析 (Fundamental Analysis)

**当前状态**: ⚠️ 仅展示营收、EPS、净利润

**优化目标**:
- 盈利能力指标: ROE、ROA、毛利率、净利率
- 成长性指标: 营收增长率、EPS增长率、净利润增长率
- 财务健康: 负债率、现金流、自由现金流
- 效率指标: 资产周转率、存货周转率

**实施优先级**: ⭐⭐⭐⭐⭐

### 维度2: 估值分析 (Valuation Analysis)

**当前状态**: ❌ 无估值数据

**优化目标**:
- 绝对估值: P/E、P/B、P/S、EV/EBITDA
- 相对估值: 与行业平均对比、与历史区间对比
- 估值百分位: 当前估值处于5年区间的什么位置
- 合理估值区间: AI估算的目标价

**实施优先级**: ⭐⭐⭐⭐⭐

### 维度3: 行业对比 (Peer Comparison)

**当前状态**: ❌ 无同行对比

**优化目标**:
- 同行业公司关键指标对比表
- 行业排名百分位 (P/E排名第几、ROE排名第几)
- 竞争优势评估 (市场份额、增长率 vs 行业)

**实施优先级**: ⭐⭐⭐⭐

### 维度4: 宏观与政策 (Macro & Policy)

**当前状态**: ❌ 无宏观分析

**优化目标**:
- 利率环境对公司的影响评估
- 监管风险识别 (反垄断、数据隐私)
- 地缘政治因素分析
- 行业政策变化追踪

**实施优先级**: ⭐⭐⭐

### 维度5: AI投资建议 (Investment Recommendation)

**当前状态**: ⚠️ 仅情绪标签 (积极/中性/消极)

**优化目标**:
- 明确投资评级: 强烈买入/买入/持有/卖出/强烈卖出
- 目标价区间: AI估算的合理价格范围
- 投资期限建议: 短期/中期/长期
- 关键催化剂: 可能推动股价的正面事件
- 风险因素: 需要关注的主要风险

**实施优先级**: ⭐⭐⭐⭐⭐

## 🏗️ 技术架构升级

### 数据层扩展

#### 新增数据表

```sql
-- 公司估值指标表
CREATE TABLE company_valuation (
  id SERIAL PRIMARY KEY,
  company_id INTEGER REFERENCES companies(id),
  market_cap BIGINT,
  pe_ratio DECIMAL(10,2),
  pb_ratio DECIMAL(10,2),
  ps_ratio DECIMAL(10,2),
  ev_ebitda DECIMAL(10,2),
  roe DECIMAL(5,2),
  roa DECIMAL(5,2),
  debt_to_equity DECIMAL(5,2),
  free_cash_flow BIGINT,
  updated_at TIMESTAMP
);

-- 行业基准表
CREATE TABLE industry_benchmarks (
  id SERIAL PRIMARY KEY,
  sector VARCHAR(50),
  metric_name VARCHAR(50),
  avg_value DECIMAL(10,2),
  median_value DECIMAL(10,2),
  updated_at TIMESTAMP
);

-- 竞争对比表
CREATE TABLE peer_comparison (
  id SERIAL PRIMARY KEY,
  company_id INTEGER REFERENCES companies(id),
  peer_symbol VARCHAR(10),
  metric VARCHAR(50),
  company_value DECIMAL(10,2),
  peer_value DECIMAL(10,2),
  percentile INTEGER,
  comparison_date DATE
);

-- 宏观政策表
CREATE TABLE macro_policies (
  id SERIAL PRIMARY KEY,
  policy_type VARCHAR(50),
  title TEXT,
  description TEXT,
  impact_sectors TEXT[],
  sentiment VARCHAR(20),
  effective_date DATE,
  created_at TIMESTAMP
);
```

#### 数据源扩展

| 数据源 | 用途 | 成本 |
|--------|------|------|
| Yahoo Finance API | 估值指标 (P/E, P/B等) | 免费 |
| Alpha Vantage | 技术指标、历史价格 | 免费 (5次/分钟) |
| FRED API | 宏观数据 (利率、通胀) | 免费 |
| NewsAPI | 政策新闻 | 免费 (100次/天) |

### AI分析引擎升级

#### 新Prompt架构

```
现有: 财报摘要生成
优化: 多维度投资决策分析

分析流程:
财报数据 ──┬──→ 基本面评估 ──┐
          ├──→ 估值分析 ────┼──→ 综合投资决策
          ├──→ 行业对比 ────┤
          └──→ 宏观影响 ────┘
                    ↓
         投资建议（评级+目标价+风险）
```

#### AI输出结构

```typescript
interface InvestmentAnalysis {
  investmentRating: 'strong_buy' | 'buy' | 'hold' | 'sell' | 'strong_sell';
  confidence: 'high' | 'medium' | 'low';
  targetPrice: { low: number; high: number };
  
  valuation: {
    assessment: 'undervalued' | 'fair' | 'overvalued';
    pePercentile: number; // 0-100
    reasoning: string;
  };
  
  fundamentals: {
    strengths: string[];
    weaknesses: string[];
    trend: 'improving' | 'stable' | 'deteriorating';
  };
  
  competitivePosition: {
    moat: 'wide' | 'narrow' | 'none';
    marketPosition: 'leader' | 'challenger' | 'follower';
    vsPeers: string[];
  };
  
  risks: string[];
  catalysts: string[];
  investmentHorizon: 'short' | 'medium' | 'long';
}
```
