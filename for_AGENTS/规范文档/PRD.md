# 产品需求文档 (PRD)

**文档版本:** 1.0  
**更新日期:** 2026-02-18  
**产品名称:** Earnlytics  
**产品定位:** AI 驱动的美国科技公司财报分析平台

---

## 1. 产品概述

### 1.1 产品愿景
Earnlytics 是一个专注于美国科技股的 AI 财报分析平台，为中文投资者提供及时、准确、深度的财报分析和投资建议。

### 1.2 核心价值主张
- **及时性:** 财报发布后 1 小时内提供 AI 分析
- **专业性:** 基于财务指标的专业分析框架
- **易读性:** 将复杂财报转化为中文摘要和可视化图表
- **actionable:** 提供明确的投资建议和估值参考

### 1.3 目标用户

| 用户类型 | 描述 | 核心需求 |
|---------|------|----------|
| 散户投资者 | 关注美股科技股的个人投资者 | 快速了解财报要点、获取投资参考 |
| 专业投资者 | 有财务分析基础的投资者 | 深度分析、同行对比、估值工具 |
| 财经博主 | 内容创作者 | 获取分析素材、验证投资观点 |
| 投资小白 | 刚接触美股投资的新手 | 学习财报解读、理解投资逻辑 |

## 2. 功能需求

### 2.1 核心功能模块

#### 2.1.1 财报数据中心
**功能描述:** 汇集美国科技公司的财报数据，提供结构化展示和查询

**功能列表:**
- [x] 财报列表浏览（按公司、时间筛选）
- [x] 单条财报详情展示
- [x] 关键财务指标展示（收入、EPS、增长率等）
- [x] 历史财报对比
- [x] 财报日历（upcoming earnings）

**用户故事:**
- 作为投资者，我希望查看某公司的最新财报，以便了解其业绩表现
- 作为投资者，我希望对比公司连续几个季度的财报，以便发现趋势

#### 2.1.2 AI 智能分析
**功能描述:** 利用 DeepSeek AI 对财报进行智能解读，生成中文分析

**功能列表:**
- [x] 财报摘要生成（中文）
- [x] 亮点提取（bullet points）
- [x] 风险关注点识别
- [x] 情绪分析（积极/中性/消极）
- [x] 批量财报分析（自动化）

**用户故事:**
- 作为忙碌的投资者，我希望快速获取财报的中文摘要，节省时间
- 作为非专业投资者，我希望了解财报中的潜在风险点

#### 2.1.3 投资分析工具
**功能描述:** 提供专业的投资分析工具和估值参考

**功能列表:**
- [x] 估值指标展示（PE、PB、PS 等）
- [x] 历史估值百分位（判断当前估值高低）
- [x] 同行对比分析
- [x] 财务健康度评分
- [x] 投资评级和建议
- [x] 目标价区间参考

**用户故事:**
- 作为价值投资者，我希望了解公司当前的估值是否合理
- 作为投资者，我希望对比同行业的其他公司，找到相对优势

#### 2.1.4 公司数据库
**功能描述:** 完整的美国科技公司数据库

**功能列表:**
- [x] 公司列表浏览（按行业筛选）
- [x] 公司基本信息展示
- [x] 公司搜索功能
- [x] 行业分类浏览

**用户故事:**
- 作为投资者，我希望按行业浏览公司，发现投资机会
- 作为投资者，我希望搜索特定公司，快速查看其财报

#### 2.1.5 用户系统
**功能描述:** 用户注册、登录、个性化服务

**功能列表:**
- [x] 用户注册（邮箱）
- [x] 用户登录
- [x] 个人仪表盘
- [x] 关注公司列表
- [ ] 投资笔记（待开发）

**用户故事:**
- 作为注册用户，我希望关注感兴趣的公司，方便追踪
- 作为用户，我希望有自己的仪表盘，查看关注公司的最新动态

#### 2.1.6 邮件订阅服务
**功能描述:** 邮件订阅，及时获取最新分析

**功能列表:**
- [x] 邮箱订阅（无需注册）
- [x] 财报发布提醒
- [ ] 每日/每周摘要（待开发）

**用户故事:**
- 作为访客，我希望订阅邮件，及时收到新财报的分析
- 作为订阅者，我希望选择接收频率和内容类型

#### 2.1.7 预警系统
**功能描述:** 智能预警，及时提醒重要变化

**功能列表:**
- [x] 评级变化提醒
- [x] 目标价大幅调整提醒
- [x] 估值异常提醒
- [x] 财报发布提醒
- [ ] 价格阈值提醒（待开发）

**用户故事:**
- 作为投资者，我希望在关注的股票评级变化时收到提醒
- 作为投资者，我希望在目标价大幅调整时及时了解

## 3. 非功能需求

### 3.1 性能需求
- 页面首屏加载时间 < 2s
- API 响应时间 < 500ms
- 支持并发用户 1000+

### 3.2 可用性需求
- 系统可用性 99.9%
- 支持移动端访问（响应式设计）
- 支持主流浏览器（Chrome、Safari、Firefox、Edge）

### 3.3 安全需求
- 用户密码加密存储（bcrypt）
- API 密钥环境变量管理
- 数据库访问权限控制（RLS）
- HTTPS 全站加密

### 3.4 可扩展性需求
- 支持公司数量扩展至 100+
- 支持新的数据源接入
- 支持 AI 模型切换

## 4. 数据需求

### 4.1 数据源
| 数据类型 | 来源 | 更新频率 |
|---------|------|----------|
| 财报数据 | Financial Modeling Prep (FMP) | 实时 |
| AI 分析 | DeepSeek API | 按需 |
| 公司信息 | FMP + 手动维护 | 定期 |
| 股价数据 | FMP | 实时 |

### 4.2 数据覆盖
- **公司范围:** 美国上市科技公司（当前 30 家，目标 100+）
- **财报类型:** 季报（10-Q）、年报（10-K）、业绩快报
- **历史数据:** 至少 8 个季度历史财报

---

## 5. 用户界面需求

### 5.1 设计风格
- **整体风格:** 专业、现代、简洁
- **主色调:** 深蓝紫色系（科技感）
- **暗色主题:** 默认暗色模式（适合长时间阅读）

### 5.2 关键页面
1. **首页:** 搜索入口、统计数据、财报日历、功能介绍
2. **财报详情页:** 公司信息、财务指标、AI 分析、原始数据
3. **投资分析页:** 估值分析、同行对比、投资建议
4. **公司列表页:** 筛选、排序、搜索
5. **用户仪表盘:** 关注列表、预警设置、个人中心

### 5.3 交互需求
- 流畅的页面过渡动画
- 数据加载骨架屏
- 响应式布局适配
- 键盘导航支持
